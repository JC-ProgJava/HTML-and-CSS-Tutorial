Structure

HTML tags all have opening tags <element> and closing tags </element>. All HTML & CSS code go into a file(or seperated files) that has extension names .html and .css. Default structure of HTML file:


<!DOCTYPE html>
<html lang="en">
<!--Comment-->
<!--Stylesheet(CSS) may go in link or as style tag in head-->
<head>
<meta charset="UTF-8">
<style></style>
<link rel="stylesheet" href="mystyle.css">
<title>Title</title>
</head>
<body>
</body>
</html>

Text Decoration

Use tags <mark>, <em>, <b>, <strong>, <i>, <small>, <del>, <ins>, <sub> and <sup> to format elements to display special types of text.

<mark>Highlighted text</mark>
<b>Bold text</b>
<strong>Strong text</strong>
<i>Italic</i>
<small>Small text</small>
<del>Deleted Text</del>
<ins>Inserted Text</ins>
<em>Emphasis</em>
Hello <sub>Subscript</sub>!
Hello <sup>Superscript</sup>!

Colors and Styling

Style and color HTML content using the style attribute like this:

<h3 style="background-color: tomato; color: blue;">Hello</h3>

Hello

Syntax structure:

selector { property: value; }

You can also style text with the <style> tags in the head or in a external stylesheet linked to the HTML document, as shown in the Structure section.
Color formats used in HTML are:

  • Hexadecimal values
  • RGB values
  • Name values(like Blue, Red, Black, Tomato...)

Links and Images

Links takes the form <a href="https://www.google.com/"> in HTML.
Links can have additional properties like:

Property Values
target
  • _blank (open in new tab)
  • _parent (opens on this tab)
  • _self (opens on this tab)
  • _top (opens on this tab)

They can also link to different parts of this webpage like the link in the Colors and Styling section.

<a href="#styling">Hello!</a> <!--Uses the id tag with '#' to link to section-->
<h2 id="styling">Colors and Styling</h2>

Images can also be inserted using the <img> tag:

<img src="image.jpg">

Note that the image tag does not require a closing tag. You can also add a alternate text if the image does not load in the website like this:

<img src="image.jpg" alt="Image failed to load">

Output:

Images can also link to other places when enclosed in <a> tags.

    

Tables and Lists

Tables are created with the <table> tag. Each row is wrapped in <tr>(table row) tags and their content is wrapped in <th>(table head) and <td>(table data) tags.

                                                    
Header Name
Row 1 John
Row 2 Mary
        
         <div>
      <table>
        <tr>
          <th>Header</th>
          <th>Name</th>
        </tr>
        <tr>
           <td>Row 1</td>
           <td>John</td>
        </tr>
        <tr>
          <td>Row 2</td>
          <td>Mary</td>
        </tr>
      </table>
    </div>         

Lists are created with the <ol>(ordered list/numbered list) or <ul>(unordered list/bullet list) tags. Each value is wrapped in <li>(list item) tags.

  1. No. 1
  2. No. 2
  3. No. 3
    4.     
  • Bullet 1
  • Bullet 2
  • Bullet 3
